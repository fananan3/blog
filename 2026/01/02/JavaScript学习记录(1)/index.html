本章学习了js的变量声明方式和作用域，数据类型和结构，数据类型的字面量

变量声明方式

var，let,const

全局变量var：声明一个变量可将其初始化一个值
局部变量let：声明一个块级作用域的局部变量，可选择蒋其初始化一个值
局部变量const：声明一个块级作用域的只读命名常量
块级作用域就是被{}包起的区域

变量提升

var

因为var变量可提升，你可以在任何地方引用var，但是只有该变量的声明和默认初始化会被提升而不是它的被赋的值

let,const

let,const不会被提升，会提示引入错误

常量

const

常量的值不能被重新赋值，但是可以被修改常量定义的对象的属性

数据结构和类型

数据类型 8种

Boolern true和false
null 一个表示空值的特殊关键字
undefined 一个未定义的顶级属性
Number 整数或浮点数
BigInt 任意精度的整数
String 字符串
Symbol 其实例是唯一且不可变的数据类型
Object 对象 特殊的数据类型

数据类型的转换

JavaScript是动态类型语言，在赋值时可以不用声明数据类型可以根据需要自动转换数据类型

这不会出错

数字 “+” 运算符

在使用+运算符的表达式种涉及到字符串和数字时，通过+都会转变为字符串

在使用其他字符串的时候就不会转变为字符串

字符串转为其他数字

parseInt(),parseFloat()

在使用parseInt()时总是加上进制参数

字面量

数组字面量

数组字面量用 [] 表示

布尔字面量

布尔类型有true，false两种字面量

整数字面量

浮点数字面量

对象字面量

对象字面量里还可以套用对象字面量

对象字面量的属性的名字可以是任何字符串，包括空串，如果属性名不合理必须用单引号包住

用单引号包住的属性名不能通过.来调用

而是通过方括号表示法[] 来访问
