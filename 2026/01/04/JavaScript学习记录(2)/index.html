本章学习了条件语句和异常处理语句

条件语句

if…else,switch case,break

在if…else中，注意判断if条件中的值的判断是否为真值还是假值

假值

下面这些值求值为 false（也叫做假值）：

false
undefined
null
0
NaN
空字符串（""）

所有其他的值——包括所有的对象——在被传递给条件语句时会求值为 true。

异常处理语句

throw

使用throw语句可以抛出异常值，可以抛出数值类型，布尔类型，对象类型，字符串类型

try…catch

try语句中写可能会出现异常的语句，如果出现异常就转到catch块来写对这个异常的处理语句，如果没有出现异常，catch快会被跳过，除此之外还有finally块无论是否出现异常finally块都会执行，始终执行在try…catch之下，和其他语句之上。
catch块的标识符可以用存储抛出的异常值，并且标识符只存在catch块之内，catch块运行结束标识符不可用
